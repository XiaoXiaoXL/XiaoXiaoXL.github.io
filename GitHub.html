GitHub基础使用与个人博客上传

​ 上次给大家介绍了搭建基于python的个人博客，今天主要说搭建好的博客如何上传到Git中，先来学习一些git的基础语法，

Git基础语法：

git init 初始化仓库

git add . 将所有文件添加到仓库

git commit -m "first commit" 提交项目并备注整改内容

git remote add origin https://github.com/XiaoXiaoXL/XiaoXiaoXL.github.io.git push的网址及远程仓库

git remote -v 查看git的远程连接

git push -u origin master 将项目push到远程仓库

git remote remove origin 删除本地指定的远程地址

个人博客如何上传到git

首先需要在GitHub中创建一个静态内容空间，格式为账号名.github.io,注意账户名需要和登录的git账户名相同

现在开始上传git的操作啦(终于进入正题了...)

进入博客中output所在位置，打开Git Bash Here 命令行 

git init  
git add  . 
git commit -m "first commit"   
git remote add origin  `账号名.github.io 网址`
git push

这样你的博客就push上Git啦，在浏览器输入你的 https://你的账号.github.io/ 就可以访问了。
