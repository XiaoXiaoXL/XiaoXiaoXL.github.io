打造个人博客

​ 其实大部分有技术追求的开发者都有写技术博客记笔记的良好习惯， 好处就不用说啦，很多，今天呢给大家介绍一款基于python的静态网页生成器Pelican，它是一位法国开发者开发的，非常好用，它的每一篇文章会事先渲染成HTML静态文件，访问速度非常快 。不啰嗦，上操作：

首先安装pelcan和markdown 

pip install pelican  markdown --upgrade

然后建立一个文件夹用来作为博客目录 

mkdir my_blog

cd到该目录下，运行命令生成博客框架 

pelican-quickstart

安装过程过，会有很多配置项，如果不知道怎么填就直接忽略或者选择n，这些配置项在后面再添加的。

创建完成后生成的目录结构如下：

文章保存在content目录下，通常情况我们写文章采用markdown语法，所以，现在我们在content目录创建一篇文章one.md 

Title: 潇潇的技术博客
Date: 2019-08-08 10:20
Modified: 2019-08-08 19:30
Category: 技术
Tags: pelican, publishing
Slug: my-super-post
Authors: 潇潇

    开始我的博客记录，第一篇

然后运行命令生成静态页 

pelican content

执行完成后，output 目录会多出一些文件，markdown文件最终也会转换成html文件放到该目录下。

​ 使用像 Pelican 这样流行的开源软件的好处之一是，非常多的用户会做出更改并将其贡献给项目。许多都是以主题形式贡献的。 网站的主题会设置颜色、布局选项等。尝试一个新主题非常容易，你可以在http://www.pelicanthemes.com/预览其中的许多内容。

​ 在博客根目录下，将主题克隆下来

git clone https://github.com/getpelican/pelican-themes.git

执行后根目录下会出现一个pelican-themes文件，里面有很多别具风格的主题，大家尽情挑选自己满意的

打开pelicanconf.py配置文件，更改或添加THEME为自己喜欢的主题 

THEME = 'pelican-themes/waterspill'

注：首次搭建博客是没有THEME的需要自己添加

​ THEME是主题的路径，直接修改/后面的文件夹名称就可以完成修改主题

再次生成静态页，启动服务

pelican content
pelican --listen

每次写新的文章都需要生成静态页面，需要访问8000端口查看自己的博客，8000端口是不可以占用的

访问http://localhost:8000

到这里基于python的博客就搭建成功啦！！！

大家将output文件夹push到github上就可以用网址访问了 ，具体操作大家参考我的下一篇文章
